# Ropher

A minimalist Pomodoro timer with visual time tracking.

![Ropher Screenshot](docs/ropher.jpg)

## Features

- **Flexible intervals** — 5, 15, 30, 35, or 45 minute sessions
- **Task categories** — Color-coded work types with time summaries
- **Visual timeline** — See your day's work at a glance
- **Goal setting** — Set a focus goal before each session
- **Audio alerts** — Notification when time expires
- **Fullscreen mode** — Distraction-free timer view
